Noter Help

How to save a Note
  1. Go to a website that you want to take a note of.
  2. Select the text you want to save.
  3. Right click the selected text.
  4. Click the "Add New Note" context item from the menu.

Note: Will not work inside an IFrame.*

How to Use Notes

If you have any notes, you will see them inside the main GUI. You can hover over any button to see more information.

  1. Note Title: The title of the note.
  2. Note Buttons: Use these to manipulate the note.
    1. Change Text: Used to change the title / note body text. When clicked, the icon will change to Save and the editable areas will recieve a black background.
      1. Save: Save the text that is in the title and body sections.
    2. Copy: Copy the note body text to the system clipboard. (To paste it somewhere else)
    3. Collapse: Collapse the note body. Will change the icon to the expand icon when clicked.
      1. Expand: Expand the note body. Will change to collapse icon when clicked.
    4. Delete: Delete this note.
  3. Note Body: This is the text you added.
  4. URL: This is the URL of the page that you used to add the note. You can click this to go back to the page.
Tools

Tools Menu


These are actions you can perform on all the notes:


  1. Lock/Unlock Tools: Lock or Unlock Tool menu to the screen - When locked, will stay visible while scrolling. (locked by default)
  2. Help: Click to see this menu.
  3. Expand All: Expand all notes - same as clicking "Expand" button on all notes.
  4. Collapse All: Collapse all notes - same as clicking "Collapse" button on all notes.
  5. Delete All: Delete all your notes.
